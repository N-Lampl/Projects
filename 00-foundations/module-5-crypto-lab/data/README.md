# Data

This lab needs **no external dataset**. Everything is synthetic and generated
at runtime:

- The ECB-penguin image is a procedurally generated grayscale silhouette
  (`crypto_lab.image.make_penguin`) — deterministic, in-memory, never written
  to disk by default.
- The "breached password database" is a small synthetic user list defined in
  `scripts/run_crypto_lab.py`.
