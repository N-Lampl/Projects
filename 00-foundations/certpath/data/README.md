# data/ (git-ignored)

This project needs **no dataset**. Its only "data" is the human-authored
[`../syllabus.json`](../syllabus.json) — the Security+ SY0-701 domain → repo-module mapping — which
is committed because it *is* the artifact, not a downloaded corpus.

- **Source material:** CompTIA Security+ SY0-701 official exam objectives (publicly published exam
  blueprint; domain names and weights only — no copyrighted question content is reproduced).
- **License:** the objectives/blueprint are CompTIA's; this repo records only the domain titles and
  their published percentage weights for study-planning, which is fair-use referencing.
- **Download:** none required. To re-read the authoritative objectives PDF (free):
  visit <https://www.comptia.org/certifications/security> and download the SY0-701 objectives.

Nothing is written to this folder at runtime; it exists only to document provenance.
